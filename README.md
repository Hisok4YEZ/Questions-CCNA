# Quiz CCNA 2 (Modules 5-6 et 7-9)

Petit quiz d'entraînement CCNA 2 en HTML/CSS/JS.

## Structure

- `index.html` : affichage de l'interface
- `styles.css` : style du quiz
- `app.js` : logique du quiz (navigation, score, correction)
- `questions-5-6.js` : banque de questions modules 5-6
- `questions-7-9.js` : banque de questions modules 7-9

## Source des questions

Les questions ont été tirées du site : [ccnareponses.com](https://ccnareponses.com/)

- Modules 5-6 : <https://ccnareponses.com/modules-5-6-examen-de-reseaux-redondants-reponses/>
- Modules 7-9 : <https://ccnareponses.com/modules-7-9-examen-des-reseaux-disponibles-et-fiables-reponses/>

## Lancer le quiz

Ouvrir `index.html` dans un navigateur.
